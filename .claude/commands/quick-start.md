# Quick Start Guide

Welcome to kron-claude! This repository contains everything you need to set up Claude Code with powerful subagents on any machine.

## Available Commands

Display this helpful overview:

```
kron-claude Quick Start
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SETUP COMMANDS
──────────────
/buildwithagents    Interactive wizard to install and configure
                    subagents tailored to your project type

/init-project       Initialize a new project with CLAUDE.md
                    and recommended configuration

MANAGEMENT COMMANDS
───────────────────
/list-agents        Browse all available subagents from the
                    community repository

/sync-agents        Update installed agents to latest versions
                    while preserving customizations

/update             Update Claude Code itself to the latest
                    version and sync configurations

GETTING STARTED
───────────────
1. Clone this repo to your machine
2. Run /buildwithagents in your target project
3. Follow the interactive prompts
4. Start building with AI-powered assistance!

AGENT CATEGORIES
────────────────
• Core Development    - fullstack, frontend, backend, API
• Infrastructure      - DevOps, cloud, Kubernetes, Terraform
• Quality & Security  - testing, security audits, code review
• Data & AI           - ML, data engineering, LLM architecture
• Developer Experience - docs, refactoring, tooling

For more details: /list-agents
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## Tips

- All commands are interactive and will guide you through each step
- Agents are fetched from the community repository and can be customized
- Your customizations are preserved during syncs
- Use CLAUDE.md in each project to give context to your agents
